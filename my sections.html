Cascading Style Sheets (CSS) are an important way to control how ones web pages looks. CSS controls the fonts, text, colors, backgrounds, margins, and layout. CSS offers several significant advantages over alternative approaches to web design.

Here are the differnce between Padding, Margins and Borders;

  • Padding:is a CSS property that defines the space between an element content and its border (if it has a border).
  • Margins: is a CSS property that shows the space outside the border.
  • Borders:is a CSS property that goes around a div/element of the content.
